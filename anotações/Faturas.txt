porém pra gerar pdf eu vou precisar adicionar mais uma bilioteca ao laravel no caso 
eu vou usar dompdf conforme a documentação "https://laravel.com/docs/12.x/billing#invoices" 
sugere.

1 - foi criada uma rota de dowload de fatura 

    Route::get('/invoice/{id}',[MainController::class,'invoiceDownload'])->name('invoice.dowload');

2 - depois um metodo que faz o dowload da fatura quando a rota é acionada 
    
     public function invoiceDownload($id){
    
      return auth()->user()->invoiceDownload($id);
         
    }

3 - como o metodo "invoices()" retorna varias faturas dependendo do tanto de faturas que cliente tiver

     @foreach ($invoices as $invoice)
